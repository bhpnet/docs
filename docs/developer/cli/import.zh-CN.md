## import

导入区块链数据

import 命令从 RLP 编码的表单导入块。表单可以是一个包含多个 RLP 编码块的文件，也可以使用多个文件。

如果只使用一个文件，导入错误将导致失败。如果使用了多个文件，即使发生单个 RLP 文件导入失败，也将继续处理。

### 命令

```shell
import [command options] [arguments...]
OPTIONS
--datadir  value  数据主目录
--syncmode value  同步模式 ("fast", "full", or "light") (默认: fast)
```
